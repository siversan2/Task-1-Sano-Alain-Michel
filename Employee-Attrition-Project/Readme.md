# Employee Attrition Analysis & Prediction

## Project Overview
This project focuses on analyzing employee attrition using Data Science and Machine Learning techniques. The objective is to identify factors that contribute to employee turnover and build predictive models that can help organizations improve employee retention.

The project was completed as part of a Data Science Internship task.

---

## Objectives
- Understand the employee attrition dataset
- Clean and preprocess the data
- Perform Exploratory Data Analysis (EDA)
- Visualize employee trends and patterns
- Build machine learning models to predict employee attrition

---

## Dataset Information
The dataset contains HR-related employee information such as:
- Age
- Department
- Job Role
- Monthly Income
- Job Satisfaction
- Overtime
- Years at Company
- Attrition Status

Dataset Source:
IBM HR Analytics Employee Attrition Dataset

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## Project Workflow

### 1. Data Collection & Understanding
- Loaded the dataset
- Checked dataset structure
- Identified columns and data types
- Understood dataset features

### 2. Data Cleaning & Preprocessing
- Checked missing values
- Removed duplicates
- Encoded categorical variables
- Prepared data for machine learning

### 3. Exploratory Data Analysis (EDA)
- Analyzed attrition distribution
- Identified trends and patterns
- Explored relationships between variables
- Detected important attrition factors

### 4. Data Visualization
Created visualizations including:
- Count plots
- Histograms
- Box plots
- Heatmaps
- Correlation analysis

### 5. Machine Learning Modeling
Built predictive models:
- Logistic Regression
- Random Forest Classifier

Evaluated models using:
- Accuracy Score
- Classification Report
- Confusion Matrix

---

## Key Insights
- Employees working overtime had higher attrition rates.
- Lower job satisfaction increased the likelihood of leaving.
- Lower monthly income was associated with higher turnover.
- Employees with fewer years at the company were more likely to leave.

---

## Results
The Random Forest model achieved strong predictive performance in identifying employees likely to leave the organization.
